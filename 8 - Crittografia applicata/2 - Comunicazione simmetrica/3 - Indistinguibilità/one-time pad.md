## Funzionamento

L'*one-time pad* si basa sull'invertibilità della funzione logica [[XOR]].

```python
def onetimepad(key):
	def encrypt(plaintext) -> ciphertext:
		return key ^ plaintext
	def decrypt(ciphertext) -> plaintext:
		return key ^ ciphertext
	return encrypt, decrypt
```
