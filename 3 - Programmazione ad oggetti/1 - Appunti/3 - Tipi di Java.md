# Tipi del Java

## Primitivi

- byte (8 bit)
- short (16 bit)
- int (32 bit)
- long (64 bit)
- float (IEEE 754 | 32 bit)
- double (IEEE 754 | 64 bit)
- char (UTF-16)
- boolean (1 bit)

## Wrapper

Esistono classi Wrapper che trasformano i dati primitivi in delle classi:

- Byte
- Short
- Integer
- ...
